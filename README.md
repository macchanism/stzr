# stzr
### 実行

stzrを開ける
```
./stzr
```

![](sample.gif)

#### ラベルオプション

| option |      label      |
| ------ | --------------- |
|        |                 |
|   -F   |   GRAPEFRUIT    |
|   -L   |      LEMON      |
|   -A   |      APPLE      |
|   -O   |     ORANGE      |
|   -Y   |      YUZU       |
|   -K   |   KANJUKU-UME   |
|   -C   | CITRUS DEPRESSA |
|   -B   |  BITTER  LEMON  |
|   -P   |      PEACH      |
|   -G   |      GRAPE      |
|   -D   |       DRY       |
|   -M   |      MELON      |

#### その他のオプション

プルタブを自ら開ける
```
./stzr -p
```

ロング缶
```
./stzr -l
```

スペースシャトル (350ml)
```
./stzr -s
```

ロケット (ロング缶)
```
./stzr -r
```
